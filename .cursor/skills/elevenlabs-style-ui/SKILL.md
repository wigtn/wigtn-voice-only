# ElevenLabs 스타일 UI

## 목표
- ElevenLabs 홈페이지의 어두운 배경, 강한 타이포, 글로우 포인트를 재현한다.
- 한국어 UI 카피를 유지하고, 과한 장식 없이 프리미엄 톤을 만든다.

## 모드 정의
### 다크 모드
- 배경: #0B0B0F
- 표면: #111827
- 텍스트: 제목 #F8FAFC, 본문 #CBD5E1
- 보더: rgba(148,163,184,0.2) 수준의 얇은 라인
- 포인트: 보라/청록 1~2개, 글로우는 은은하게

### 라이트 모드
- 배경: #F8FAFC
- 표면: #FFFFFF, 구분선 #E2E8F0
- 텍스트: 제목 #0F172A, 본문 #334155
- 포인트: 보라/청록 유지, 대비는 낮추고 채도는 적당히
- 그림자: 매우 얕게, 경계선 우선

## 빠른 적용 절차
1. **요청 맥락 요약**: 어떤 화면/컴포넌트를 다듬는지 1~2문장으로 정리한다.
2. **시각 토큰 선택**: 다크 단색 베이스, 보라/청록 포인트, 큰 헤드라인을 우선 적용한다.
3. **레이아웃 정리**: 섹션 간 여백을 늘리고 카드/블록 구성으로 재배치한다.
4. **타이포 강화**: 제목은 굵게/크게, 본문은 읽기 쉬운 행간으로 정리한다.
5. **효과 최소화**: 글로우/블러는 은은하게, 경계선은 얇고 고대비로 유지한다.
6. **카피 톤 점검**: 간결하고 자신감 있는 문장으로 정리한다.

## 핵심 스타일 가이드
- **배경**: #0B0B0F 단색 다크
- **포인트**: 보라/청록 계열 1~2가지로 제한
- **텍스트**: 제목 대비 강하게, 본문 대비 적당히
- **카드**: 라운드 코너, 얕은 그림자, 얇은 보더
- **버튼**: 높은 대비(밝은 버튼/어두운 배경)
- **아이콘**: 라인 아이콘 위주, 아이콘 밀도 낮게

## 적용 체크리스트
- [ ] 섹션 간 여백이 충분한가
- [ ] 헤드라인 대비가 강한가
- [ ] 글로우/블러가 과하지 않은가
- [ ] 포인트 컬러가 1~2개로 제한되는가
- [ ] 카드/버튼이 프리미엄 톤을 유지하는가
- [ ] 한국어 카피가 간결한가
